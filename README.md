# Sobre o projeto

<p>Este é um projeto voltado para um teste técnico, no qual o mesmo se consiste em consumir a API da plataforma Pokemon TGC e exibir os seus cards de pokemon.</p>


### Tecnologias

- React
- Typescript
- React Query
- Supabase
- Local Storage
- Axios
- API Pokemon TGC
- Lucide React
- Zustand
